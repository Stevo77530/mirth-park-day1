# MIRTH Park — Day One

Isometric prototype for a dark satirical theme park management sim with an unhinged DCC-style AI narrator.

## Features
- Isometric grid view
- Guest agents with stats (Joy, Fear, Escape Drive, Suspicion, etc.)
- Build mode (paths, rides)
- MIRTH AI with gaslighting, achievements, direct address, and showrunner energy
- Visual horror (vignette, watching guests, degradation)
- Retention events and disappearances

Run by opening `index.html` in a browser.

Next steps: More DCC-style features, escalation, quests, and visual polish.
